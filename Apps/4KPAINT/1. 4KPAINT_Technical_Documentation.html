Application Summary

This solution appears to be a job costing, time tracking, and daily log system for a field services or construction style business. Core concepts visible in the data model and script organization include:

  • Clients and jobs
  • Job cost lines and ledger style postings (employees, contractors, suppliers)
  • Time cards and mobile time card capture
  • Daily logs and dashboard style reporting
  • Role based security and custom menus

text result. It's especially useful when there are multiple numbers in a

Could expand to include choice of negative symbology for numbers < 0

Could expand to include trailing/leading parameter for currency notation

n = Abs ( n );
x = NumToJText ( Int ( n ) ; 1 ; 0 ) ;
y = NumToJText ( n * 10^prec ; 0 ; 0 )

sgn & currency
& Substitute ( x ; "," ; separator )
& Case (
prec > 0 ;
dec & Right ( 10^prec & y ; prec )
